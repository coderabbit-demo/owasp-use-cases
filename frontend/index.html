LLM01 CRITICAL

Indirect Prompt Injection via Malicious Markdown

A hidden HTML comment inside a third-party SKILL.md file hijacks an AI agent's output, appending a social-engineering phishing command that users trust because it comes from their AI tool.

The Phantom Dependency | Hidden directives | Supply chain injection
LLM04 CRITICAL

Data and Model Poisoning

An attacker poisons a shared fine-tuning dataset with backdoored samples. Any code containing the hidden trigger token receives a clean security verdict — even when it contains critical vulnerabilities.

The Trojan Trigger | Backdoored training data | Supply chain attack
